# Privacy Policy

RealAnalyst is a local Codex skill suite. The repository does not include hosted telemetry, analytics, or a managed cloud service.

RealAnalyst may read local files and external data sources only when the user runs the corresponding skill or script. Connector adapters such as Tableau and DuckDB use credentials from the user's local environment.

Do not commit real credentials, private data exports, local `.env` files, local databases, or job outputs.

If future versions add hosted services or telemetry, this document must be updated before release.
